# Enterprise Identity & Subscription Platform

This project simulates an enterprise backend platform used in Telco/Banking environments.

## Core features
- User management
- Subscription activation
- Asynchronous provisioning via Kafka
- REST and SOAP APIs
- OAuth2 / JWT security
- Full observability and CI/CD

## Goal
Build an end-to-end backend system covering development, security, DevOps and cloud deployment.
